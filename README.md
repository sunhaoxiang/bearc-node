# bearc-node

> 使用 nodejs + express + mongoDB 开发的后台管理系统服务端

> 客户端源码地址: [bearc-vue](https://github.com/sunhaoxiang/bearc-vue)

## 功能

- 注册、登录，对密码进行加盐处理
- 修改密码
- json-web-token权限验证
- 历史上的今天接口
- 商品、客户以及收入列表的添加、删除和修改
- 使用helmet隐藏http header关键信息，提高安全性
